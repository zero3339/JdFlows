## 🚀 快速开始

```bash
# 1. 克隆项目
git clone https://github.com/zero3339/JdFlows.git
cd JdFlows

# 2. 安装依赖
pip install -r requirements.txt

# 3. 安装Playwright浏览器
playwright install chromium

# 4. 运行应用
python main.py
```
